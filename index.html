Aufgabenvorstellung

Es werden zwei Beispiele präsentiert, jeweils eine ungesicherte Website, dann die gesicherte. Gewählt worden sind zwei Kategorien der OWASP Top 10 Liste, "Injection" und "Cryptographic Failures".

Die Website wurde mit PHP und HTML5 erstellt. Außerdem verwendet die Website das Framework Bootstrap für das Design. Die Website bildet einen kleinen Ausschnitt für einen Mitarbeiterbereich ab für einen Verlag ab. Es werden auf der Homepage die Liste an Büchern präsentiert, die auch Kund:innen sehen können. Mitarbeiter:innen jedoch können Bücher löschen und hinzufügen. Des Weiteren wird auch ein Login / User Wechsel angeboten.

Injections

Mit diesem Beispiel wollen wir Ihnen zwei Injections präsentieren und wie man sich dagegen schützt.
Bei einer Injection handelt es sich um ein Cross-Site-Scripting, genauer um eine Stored XSS. Bei der zweiten Injection um eine SQL Injection. Das Cross-Site-Scripting wird im Bereich des Buch-Hinzufügens und im Bereich für die Liste der Bücher präsentiert. Die SQL Injection im Loginbereich.

Cryptographic Failures

Mit diesem Beispiel wollen wir Ihnen die Verwendung von veralteten Verschlüsselungen und dessen Auswirkungen präsentieren. In PHP gibt es einige Funktionen für Verschlüsselungen, die weiterhin Verwendung finden, vor allem in alten Websites, jedoch offiziell als deprecated gelten. Für dieses Beispiel wurde der Login Bereich verwendet.
